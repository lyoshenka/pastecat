Set up an alias for easy usage:

  $ alias pcat='curl -F "{{.FieldName}}=<-" {{.SiteUrl}}'

Upload a new paste:

  $ echo foo | pcat
  {{.SiteUrl}}/a63d03b9

Fetch it:

  $ curl Ì£{{.SiteUrl}}/a63d03b9
  foo

Maximum size per paste is {{.MaxSize}}. Pastes will be deleted after {{.LifeTime}}.
