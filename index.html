Upload a new paste:

  $ echo foo | curl -F 'paste=<-' {{.SiteUrl}}
  {{.SiteUrl}}/a63n03rp

Fetch it:

  $ curl Ì£{{.SiteUrl}}/a63n03rp
  foo

The paste will be deleted after {{.LifeTime}}.

Alternatively, you can use the web form or a shell function:

  pcat() {
    if [ -t 0 ]; then
      [ $# -gt 0 ] || return 1
      cat "$@" | curl -F 'paste=<-' {{.SiteUrl}}
    else
      curl -F 'paste=<-' {{.SiteUrl}}
    fi
  }

This will allow for easier usage:

  $ pcat file
  $ echo foo | pcat

It also supports basic MIME type inference, so you may upload small images or
PDF documents.
