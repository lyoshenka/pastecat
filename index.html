Upload a new paste:

  $ echo foo | curl -F 'paste=<-' {{.}}
  {{.}}/a63n03rp

Fetch it:

  $ curl Ì£{{.}}/a63n03rp
  foo

Alternatively, you can use the web form or a shell function:

  pcat() {
    if [ -t 0 ]; then
      [ $# -gt 0 ] || return 1
      cat "$*" | curl -F 'paste<-' {{.}}
    else
      curl -F 'paste<' {{.}}
    fi
  }

This will allow for easier usage:

  $ pcat file
  $ echo foo | pcat
