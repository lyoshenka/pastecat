Upload a new paste:

  $ echo foo | curl -F '{{.FieldName}}=<-' {{.SiteUrl}}
  {{.SiteUrl}}/a63d03b9

Fetch it:

  $ curl Ì£{{.SiteUrl}}/a63d03b9
  foo

Alternatively, you can use the web form or a shell function:

  pcat() {
    if [ -t 0 ]; then
      [ $# -eq 1 ] || return 1
      curl -F "{{.FieldName}}=<-" {{.SiteUrl}} < "$1"
    else
      curl -F "{{.FieldName}}=<-" {{.SiteUrl}}
    fi
  }

This will allow for easier usage:

  $ pcat file
  $ echo foo | pcat

Maximum size per paste is {{.MaxSize}}. Pastes will be deleted after {{.LifeTime}}.
